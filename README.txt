Proyecto Cootransporte

Este proyecto es un sistema básico en Java con conexión a base de datos MySQL.

Funcionalidades

Conexión a MySQL con JDBC.

Operaciones CRUD:

Insertar

Consultar

Actualizar

Eliminar

Herramientas utilizadas

Java (NetBeans 8.2)

MySQL Workbench

GitHub (para versionamiento)

Ejecución

Crear la base de datos cootransporte en MySQL.

Cambiar usuario y contraseña en el archivo ConexionBD.java.

Ejecutar MainApp.java para probar el CRUD desde consola.
